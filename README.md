# Obsidian Flash

Enables [flash.nvim](https://github.com/folke/flash.nvim)-like behavior while using Vim mode in Obsidian. 

By default, "s" will trigger flash from normal or visual mode, but you can change the key in the plugin settings. If you use the [vimrc plugin](obsidian://show-plugin?id=obsidian-vimrc-support), you can leave the keymap setting blank and instead map `":flash<CR>"` in your vimrc. 

If you use vimrc and choose to map flash using the built-in settings instead of in your vimrc, just make sure the key you set doesn't conflict with a mapping in your vimrc.
